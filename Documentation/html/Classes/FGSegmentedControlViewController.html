@property (nonatomic, assign) NSUInteger selectedIndex

Discussion

Setting this property changes the selected view controller to the one at the designated index in the viewControllers array. The default value of this property is 0.

viewControllers

An array of the root view controllers displayed by the segmented control.

@property (nonatomic, copy) NSArray *viewControllers

Discussion

When configuring a segmented control controller, you can use this property to specify the content for each segment of the segmented controller. The order of the view controllers in the array corresponds to the display order in the segmented control. Thus, the controller at index 0 corresponds to the left-most segment, the controller at index 1 the next segment to the right, and so on. The default value of this property is nil.
